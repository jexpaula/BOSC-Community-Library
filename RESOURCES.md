# BOSC Community Library - Resource Index

## Featured Educational Resources

The BOSC Community Library curates high-quality open educational resources for developers, educators, and students. Below is our indexed collection:

### Governance & Legal
- [Open Source Initiative (OSI)](https://opensource.org) - Open source license certification body
- [Apache Software Foundation](https://apache.org) - Community-driven open-source projects
- [Free Software Foundation](https://fsf.org) - Defender of software user freedoms
- [Creative Commons](https://creativecommons.org) - Standardized content sharing licenses
- [Digital Public Goods Alliance](https://digitalpublicgoods.net) - UN-endorsed digital public goods

### Documentation & Learning
- [Mozilla Developer Network (MDN)](https://developer.mozilla.org) - Web technology documentation
- [GitHub Docs](https://docs.github.com) - Version control and collaboration guides
- [Wikibooks Open Source Collection](https://en.wikibooks.org/wiki/Open_Source) - Free open-source textbooks
- [CodeRefinery](https://coderefinery.org) - Research software best practices

### Community & Training
- [Open Knowledge Foundation](https://okfn.org) - Open knowledge advocacy
- [The Linux Foundation](https://linuxfoundation.org) - Open-source ecosystem support
- [Bugema University Open Source Initiative](https://bugemauniv.ac.ug/open-source) - Local educational OSS hub

## About This Library

This resource database is a **living document**. If you know of a high-quality open educational resource that should be included, please [open an issue](https://github.com/jexpaula/BOSC-Community-Library/issues/new/choose) or submit a pull request.

See `DATABASE.json` for the machine-readable version suitable for integration into other platforms.
