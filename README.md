# BOSC Community Library

[![License: Apache-2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LEGAL/LICENSE)
[![GitHub contributors](https://img.shields.io/github/contributors/jexpaula/BOSC-Community-Library)](https://github.com/jexpaula/BOSC-Community-Library/graphs/contributors)
[![GitHub issues](https://img.shields.io/github/issues/jexpaula/BOSC-Community-Library)](https://github.com/jexpaula/BOSC-Community-Library/issues)

A curated, community-driven digital library built and maintained by the Bugema Open Source Community (BOSC). An open platform for sharing, discovering, and managing educational resources.

## Features

- **Searchable Resource Database** – Quick access to vetted open educational resources via a structured JSON index.
- **Multi-language Support** – Resources organized for English, Luganda, Swahili, and future extensions.
- **Transparent Governance** – All contributions undergo peer review; see [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) and [CONTRIBUTING.md](CONTRIBUTING.md).

## Quick Start

Clone the repository and explore the `DATABASE.json` and `RESOURCES.md` for our indexed content.

```bash
git clone https://github.com/jexpaula/BOSC-Community-Library.git
```
